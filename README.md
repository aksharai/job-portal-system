# Job Portal System (MERN)

## Overview
A simple Job Portal System built using MERN stack that allows users to view jobs, apply for jobs, and track applications.

## Features
- Job Listings & Applications
- Basic Resume Skill Matching
- Candidate Dashboard
- Employer Dashboard 
- Admin Panel (basic)

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## How to Run
Backend:
npm install
npm start

Frontend:
npm install
npm start
